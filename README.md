# New Mac OSX developer setup

One line command:

```
(mkdir -p /tmp/osx-setup && cd /tmp/osx-setup && curl -L https://github.com/christiankohler/osxsetup/archive/master.tar.gz | tar zx --strip 1 && sh ./install.sh)
```

## Features

- Installs Homebrew
- Installs apps
- Installs binaries
- Sets OS defaults
